## ルール詳細リスト（2ステップ + 無料リトライ）

P01〜P19各項目の補足テキスト・条件詳細を記載。

**P01: 利益目標**
Phase 1は6%〜12%から選択、Phase 2は5%固定。Funded口座では利益目標なし。

**P02: 最低取引日数**
各フェーズで最低3日の取引が必要。Challengeでは各日口座サイズの0.5%以上の利益が必須だが、Fundedでは指定なし。

**P03: 日次損失**
全フェーズ一律5%。ただし、ルール違反によりセカンドチャンス（無料リトライ）が発動した場合は3%に減少する。

**P04: 日次損失タイプ**
各日の終了時（EOD）における有効証拠金または残高の「いずれか高い方」を基準に計算される。

**P05: 最大損失**
5%〜10%の範囲で購入時にカスタマイズ可能。

**P06: 最大損失タイプ**
初期残高を基準とした固定値（Static）で計算される。

**P07: 一貫性ルール**
1日の利益が総利益の一定割合を超えてはならない等の一貫性ルールは設定されていない。

**P08: 利益上限**
Funded口座で獲得可能な総利益に上限は設定されていない。

**P09: 時間制限**
チャレンジ達成までの期限はなし。ただし、29日間連続で取引がない場合は非アクティブとして口座が非有効化される。

**P10: ニュース取引制限**
スイングトレードでのニュース取引は可能だが、HFTや悪意のある取引目的での利用は禁止されている。

**P11: 週末持ち越し制限**
週末をまたぐポジション持ち越しは可能。ただし週末中の「新規発注や決済」には週末取引アドオンが必要。

**P12: 日またぎ持ち越し制限**
日をまたぐポジションの持ち越しは制限なく可能。

**P13: EA（自動売買）制限**
リスク管理や分析等の裁量補助用EAのみ使用可能。完全自動のボットやHFT、アービトラージEAは禁止。

**P14: コピートレード制限**
同一人物が所有する自口座間でのみコピー可能。他者からのコピーや外部シグナル利用は禁止。

**P15: スキャルピング制限**
最低3分間のポジション保持が必要。スキャルピングアドオン購入時は最低30秒に短縮される。

**P16: ストップロス（SL）制限**
設定は必須ではないが、未設定時は最大含み損（MAE）ベースで最大リスクが計算される。

**P17: リスクルール**
Funded口座のみ適用。同じ日に1つの取引銘柄へ、日次損失制限の50%を超えるリスクを集中させることは禁止。

**P18: 最大ポジション制限**
具体的なポジション数の制限は記載なし（レバレッジや証拠金により実質的に制限される）。

**P19: 禁止行為**
| **禁止行為** | **詳細・ペナルティ** |
| ------ | ------ |
| IPアドレス共有 | 第三者との口座共有や代理取引 / アカウント強制閉鎖 |
| グループ取引 | 複数人による集団での取引 / アカウント強制閉鎖 |
| 完全自動EA利用 | 完全自動のボット等を利用した取引 / アカウント強制閉鎖 |
| 口座管理代行 | 他者に口座運用を委託する行為 / アカウント強制閉鎖 |
| ヘッジング | 複数口座間で反対売買を行う行為 / アカウント強制閉鎖 |
| HFT | 高頻度取引によるサーバー負荷 / アカウント強制閉鎖 |
| 超高速スキャルピング | 規定時間未満での決済 / ソフトブリーチまたは強制閉鎖 |
| レイテンシーアービトラージ | 価格遅延等を利用した裁定取引 / アカウント強制閉鎖 |
| 無許可の週末取引 | アドオンなしでの週末発注・決済 / アカウント強制閉鎖 |
| サーバースパム | 短期間での大量注文発注 / アカウント強制閉鎖 |
