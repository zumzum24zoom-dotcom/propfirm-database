# NotebookLM アフィリエイト調査ソース v01

**用途**: 各プロップファームのアフィリエイトプログラム情報をNotebookLMで調査・出力する際のソース
**策定日**: 2026-06-03
**バージョン**: v01

---

## 📋 このソースの役割

各プロップファームの公式サイト・アフィリエイトページをソースとして投入し、以下の情報を調査・出力する。

- アフィリエイト登録の条件・方法
- 報酬体系（単価・率・対象アクション）
- 支払い条件・方法
- 管理プラットフォーム

---

## 🗂 調査項目リスト（A01〜A08）

| ID | 項目名 | 内容 |
| :--- | :--- | :--- |
| A01 | 登録条件 | アフィリエイトになるために必要な条件。下記の登録条件分類から選択 |
| A02 | 登録URL | アフィリエイト登録ページの直リンク |
| A03 | 報酬タイプ | CPA / レベニューシェア / ハイブリッド のいずれか |
| A04 | 単価・報酬率 | 具体的な金額（$XX/件）または割合（XX%） |
| A05 | 対象アクション | 報酬が発生するトリガー（購入完了 / 登録 / Funded到達 等） |
| A06 | Cookie有効期限 | 紹介リンクのクッキー保持期間（日数） |
| A07 | 支払い条件 | 支払いサイクル・最低支払額・支払い方法 |
| A08 | 管理プラットフォーム | トラッキング・管理に使用するシステム（Impact / PartnerStack / 自社 等） |

---

## 📌 登録条件の分類

| 分類 | 定義 |
| :--- | :--- |
| A | アカウント作成のみで登録可能（プラン購入・審査・申請不要） |
| B | アカウント作成 ＋ 別途アフィリエイト申請フォームが必要 |
| C | プラン購入（自己口座保有）が必須条件 |
| D | 審査制・招待制（申請後に承認が必要） |
| E | 不明 / 記載なし |

---

## 📤 出力形式

ファームごとに以下のテーブル形式で出力すること。

### テーブル: アフィリエイト概要

**見出し**: `## アフィリエイト概要`

**説明文**: アフィリエイトプログラムの登録条件・報酬・支払い条件の一覧。

**構成**:

| 項目 | 内容 |
| :--- | :--- |
| A01: 登録条件 | A〜E ＋ 補足説明 |
| A02: 登録URL | https://... |
| A03: 報酬タイプ | CPA / レベニューシェア / ハイブリッド |
| A04: 単価・報酬率 | $XX / XX% |
| A05: 対象アクション | 購入完了 / 登録 / Funded到達 等 |
| A06: Cookie有効期限 | XX日 / 記載なし |
| A07: 支払い条件 | サイクル: XX / 最低額: $XX / 方法: XX |
| A08: 管理プラットフォーム | Impact / PartnerStack / 自社 等 |

## 📖 用語・表記ルール

### 不在表記

| 状況 | 表記 |
| :--- | :--- |
| 公式に「なし」と明記されている | 「なし（明示）」 |
| 該当項目が構造上適用されない | 「ー」 |
| 一次ソースに記述がない | 「記載なし」 |

### 報酬タイプの定義

- **CPA（Cost Per Acquisition）**: 成果1件あたり固定額が支払われる方式
- **レベニューシェア**: 紹介した顧客の購入額・売上に対して一定割合が支払われる方式
- **ハイブリッド**: CPA とレベニューシェアの組み合わせ

---

## 🚫 絶対禁止事項

- **推測・補完の禁止**: 根拠がなければ不在表記に従う。独自の解釈・推測は一切禁止
- **主観・評価表現の禁止**: 「おすすめ」「高単価」「お得」等の主観表現は禁止
- **情報の古さに注意**: アフィリエイト条件は変更されやすい。調査時点を明記すること

---

## ✅ 最終チェックリスト

出力前に以下を確認すること:

- □ A01〜A08 の全項目が埋まっている（不明は「記載なし」）
- □ 登録条件がA〜Eの分類で明記されている
- □ 報酬タイプが CPA / レベニューシェア / ハイブリッド のいずれかで明記されている
- □ 単価・報酬率に単位（$ または %）が付与されている
- □ 不在表記が「ー」「なし（明示）」「記載なし」で正しく区別されている
- □ 推測・主観表現が含まれていない
