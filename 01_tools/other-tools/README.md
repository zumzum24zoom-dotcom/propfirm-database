# AI風付箋メモ - 改善版

モダンなデザインと便利な機能を搭載した付箋メモアプリです。

## 新機能

### ✨ モダンなボタンデザイン
- フラットデザイン
- ホバーエフェクト
- アイコン付きボタン
- 洗練された配色

### 📑 Tab機能
- 複数のメモをタブで管理
- タブの追加・削除
- 各タブごとに独立したメッセージリスト

### 🖼️ 画像貼り付け
- クリップボードから画像を貼り付け
- ファイルから画像を選択
- 自動リサイズ機能

### 📋 1行ずつコピーボタン
- 各メッセージに個別のコピーボタン
- クリック時の視覚的フィードバック

## 必要なライブラリ

```bash
pip install pillow
```

## 使い方

```bash
python sticky_note.py
```

## ショートカットキー

- `Alt+N`: 新しいウィンドウを開く
- `Alt+W`: ウィンドウを閉じる
- `Alt+P`: ピン留め切り替え
- `Alt+M`: ミニモード切り替え
- `Alt+I`: IME切り替え
- `Alt+T`: 新しいタブを追加

## データ保存

データは `~/StickyNote/sticky_note_data.json` に自動保存されます。
